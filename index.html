💡 Was ist ein Prompt?

  • 📝 Definition: Ein Prompt ist eine Eingabe oder Anweisung, die Sie einer KI geben, um eine bestimmte Antwort oder Ausgabe zu erhalten.
  • 🔍 Vergleich: Man kann ihn sich wie eine Frage oder Aufgabenbeschreibung vorstellen, die möglichst klar formuliert sein sollte.
  • 🎯 Genauigkeit zählt: Je präziser Sie Ihr Anliegen beschreiben, desto besser trifft die KI das gewünschte Ergebnis.
  • ⚙️ Tool-Abhängigkeit: Je nach KI-System oder Plattform kann ein Prompt unterschiedlich aufgebaut sein – von kurzen Sätzen bis hin zu komplexen Beschreibungen.
  • 🗺️ Merke: Ein guter Prompt ist wie eine klare Wegbeschreibung für die KI, um möglichst nah an Ihr Ziel zu kommen.
